# web-studies

**Repositório de Estudos de Programação Web**

Este repositório contém exemplos e projetos de estudos relacionados à programação web utilizando Node.js, React, Vite e Next.js.

### Pré-requisitos

Certifique-se de ter o [Node.js](https://nodejs.org/) instalado em sua máquina.

### Configuração do Projeto

1. Clone este repositório:

```bash
git clone https://github.com/seu-usuario/seu-repositorio.git
```

2. Acesse o diretório do projeto:

```bash
cd seu-repositorio
```

3. Instale as dependências utilizando o npm:

```bash
npm install
```

### Estrutura do Projeto

- **`/backend`**: Contém exemplos e projetos relacionados ao desenvolvimento do backend usando Node.js.

- **`/frontend-react`**: Exemplos e projetos utilizando React.

- **`/frontend-nextjs`**: Projetos construídos com Next.js.

- **`/frontend-vite`**: Projetos utilizando Vite para desenvolvimento frontend.

### Como Usar

1. Navegue até o diretório do projeto desejado.

2. Execute o seguinte comando para iniciar o projeto:

```bash
npm start
```

3. Abra seu navegador e acesse `http://localhost:3000` para visualizar o aplicativo.
